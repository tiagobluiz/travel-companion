# Layout and Grid

## Container Widths
1. Content max width: 1200 px for desktop shells.
2. Reading/forms max width: 640 px.
3. Wide planning surfaces can exceed standard width with controlled margins.

## Grid Rules
1. Desktop: 12-column grid.
2. Tablet: 8-column grid.
3. Mobile: 4-column equivalent spacing model.

## Spacing Rhythm
Use the token spacing scale. Avoid ad-hoc pixel values.

## Section Structure
Typical page section:
1. Section header
2. Optional helper text
3. Main content container
4. Action bar

## Trip Planning Layout
1. Desktop:
- Itinerary list/board and map split-view.
2. Mobile:
- Single-column lists with tab/segment controls.
- Move actions accessible via inline menus or sheet.

## Z-Index Layers
1. Base content
2. Sticky headers/nav
3. Dropdowns
4. Modals/sheets
5. Global alerts
